# 📥 download.mizius.com

### A simple YouTube video downloader web application.

</div>

## ✨ Features
- 🎬 Fetch video information from YouTube URLs
- 📹 Download videos in multiple formats.
  - Video + Audio (MP4)
  - Video-only (merged with best audio)
  - Audio-only (M4A/WebM)
- 🍪 Cookie support for authenticated/restricted videos
- 🎨 Clean, modern UI with Tailwind CSS
- ⚡  Fast and efficient downloads using yt-dlp


## ⚖️ Disclaimer
This tool is for personal use only. Please respect copyright laws and YouTube's Terms of Service.
